# Data and Results

This directory holds experimental results generated by training runs.

## Directory structure

After running experiments, results will be organized as:

```
results/
├── cartpole_v1_nsga3_seed00/
│   ├── nsga3_seed00_pareto_front.json   # Pareto front (JSON)
│   ├── nsga3_seed00_results.pkl         # Full results (pickle)
│   ├── nsga3_seed00_history.json        # Training history
│   └── nsga3_seed00_summary.json        # Final metrics
├── cartpole_v1_moead_seed00/
│   └── ...
├── baselines/
│   └── cartpole_v1/
│       ├── weighted_sum_seed00.json
│       └── viper_seed00.json
└── figures/
    ├── pareto_cartpole_v1.png
    ├── pareto_deep_sea_treasure_v0.png
    └── ablation_cartpole_v1.png
```

## Pareto front JSON format

Each `*_pareto_front.json` file contains a list of solution dictionaries:

```json
[
  {
    "performance": 487.3,
    "interpretability": 0.823,
    "objectives": [487.3, 0.823],
    "tree_summary": {
      "depth": 3,
      "num_rules": 6,
      "features_used": [0, 2, 3],
      "state_dim": 4
    },
    "components": {
      "M_depth": 0.750,
      "M_rules": 0.710,
      "M_temporal": 0.880,
      "M_causal": 0.970,
      "composite": 0.823
    }
  },
  ...
]
```

## Reproducing results

See `experiments/reproduce_paper.sh` for the full reproduction script.
Quick start:

```bash
# Single run (CartPole, NSGA-III, seed 0)
python experiments/train.py --config configs/cartpole.yaml --algorithm nsga3 --seed 0

# Evaluate
python experiments/evaluate.py --results results/cartpole_v1_nsga3_seed00/

# All results (10 seeds × 4 environments × 2 algorithms)
bash experiments/reproduce_paper.sh
```

## Data availability

Pre-computed results for the 10-seed experiments reported in the paper are
available at the repository URL listed in the paper. Download and place them
in this `data/results/` directory.
